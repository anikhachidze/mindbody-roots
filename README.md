# MindBody Roots

A calm, content-focused health blog and affiliate landing page built with Next.js, React, JavaScript, and styled-components.

## Pages

- Home
- Blog
- Individual blog posts
- Motivational quotes
- Amazon recommendations
- About
- Health disclaimer
- Affiliate disclosure

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Production checks

```bash
npm run lint
npm run build
```

## Notes

Product links are placeholders. Replace them with real Amazon affiliate links before launch. Health content is educational and not medical advice.
